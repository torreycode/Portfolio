Who Was the MVP Flashcards

This flashcard site was designed to be able to test the users knowledge of the former NBA MVP's.
As a avid NBA fan myself, I find it important to know even the most trivial of information.
Users can press and hold to flip the cards and see which player is on the other side.
Other functions include the ability to select certain cards that the user wants to remember,
and hiding the entire selection of cards to watch a full highlight reel of the best NBA dunks of the 2018 season.

Technologies Used: HTML, CSS, Vanilla Javascript

Rick and Morty API

The Rick and Morty API is a RESTful back-end API created with Mongo and Express.
The API has the full ability to create, update, read, and destory data, allowing users to fully interact with the API.
The Rick and Mortty API displays over 150 characters from Rick and Morty.
I struggled initially with this back-end API with creating the functionality,
but since I have perservered, it has become one of my favorite projects that I have done so far.

Technologies Used: Mongo, MongoDB, Mongo Atlas, Express, Heroku, Postman
